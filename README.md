# PizzaPizza

Landing page developed as a practical activity for a junior company program, focused on learning modern web development with Next.js and external data integrations.

## Technologies
- Next.js (React)
- TailwindCSS
- TypeScript
- JSON-Server
- Prismic CMS

## Features
- Responsive design for mobile and tablet
- Menu data fetched from JSON-Server API
- Hero and features content managed through Prismic CMS
